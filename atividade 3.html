Primeiro site atividade 3

Neste site foi utilizados os breakpoints do bootstrap, containeres, row, margin e padding.

O site foi criado para implementar os conhecimentos adqueridos em sala. Foram utilizadas as classes de espaçamento para ajustar o layout, por exemplo, mt-3 para margens no topo e pb-3 para padding na parte inferior.
